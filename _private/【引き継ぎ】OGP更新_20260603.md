## やること：OGP画像の再生成とR2へのアップロード

OGP画像は Cloudflare R2（オブジェクトストレージ）に配置され、Cloudflare Worker経由で配信されています。キャラ画像を更新したので、OGP画像も作り直してR2に上げ直す必要があります。

---

## 手順

### Step 0: まずPuppeteerが動くか確認

OGP画像の生成には **Puppeteer（ヘッドレスブラウザ）** が必要です。先に動作確認してください。

```bash
cd "G:/マイドライブ/Claude Code/ウェブサイト/あそびラボ_プロジェクト/ビックファイブ診断サイト/_private"
node -e "const p = require('puppeteer'); p.launch({headless:true}).then(b => { console.log('OK'); b.close(); })"
```

これで `OK` と出れば問題ありません。エラーが出る場合は `npm install puppeteer` を実行してください。

### Step 1: 古いOGP画像を削除

**方法A（確実・全再生成）：**
```bash
# Windows PowerShell
Remove-Item -Recurse -Force "_private/ogp_images_v2/F", "_private/ogp_images_v2/M"
Remove-Item -Recurse -Force "_private/ogp_share_images_v2/F", "_private/ogp_share_images_v2/M"
```
→ 6,286枚×2を全再生成。時間はかかるが確実。

**方法B（差分のみ）：**
更新された61コードに対応するファイルのみ削除。`generate_v2.js` は resume 対応（既存ファイルをスキップ）なので、消した分だけ再生成されます。
- ただし各ソースコード（例：51111）は `toScale()` で複数の5段階コードにマップされるため、単純に61コード分だけでなく派生する全OGPコードのファイルを消す必要があります。
- 面倒なら方法A推奨。

### Step 2: カードOGP画像を再生成

```bash
cd "G:/マイドライブ/Claude Code/ウェブサイト/あそびラボ_プロジェクト/ビックファイブ診断サイト/_private"
node generate_v2.js
```

- **入力**: `images/characters/` のソース画像（更新済み）
- **出力**: `_private/ogp_images_v2/F/` `M/` （560×560 WebP）
- **hidden**: `_private/ogp_images_v2/hidden/` も生成（36枚）
- **合計**: 6,286枚
- **所要時間**: 1枚あたり2〜3秒 → 全体で3〜5時間（差分のみなら短い）
- resume対応：途中で止めても再開可能

### Step 3: シェアOGP画像を再生成

```bash
node generate_ogp_share_v2.js
```

- **入力**: Step 2のカード画像
- **出力**: `_private/ogp_share_images_v2/F/` `M/` （1200×630 WebP）
- **合計**: 6,286枚
- resume対応

### Step 4: カード画像をR2にアップロード

```bash
python _private/upload_to_r2_v2.py
```

- **R2バケット**: `bigfive-ogp`
- **アップロード先**: `M/`, `F/`, `hidden/`, `M_r0/`, `F_r0/`
- 認証情報はスクリプト内にハードコード済み

### Step 5: シェア画像をR2にアップロード

```bash
python _private/upload_share_to_r2.py
```

- **アップロード先**: `share/M/`, `share/F/`, `share/hidden/`

### Step 6: 動作確認

ブラウザで以下のURLを開いて、画像が更新されているか確認：

```
https://bigfive.jr-genius.jp/ogp-image?code=51111&gender=F&rarity=r5
https://bigfive.jr-genius.jp/ogp-image?code=51111&gender=F&rarity=r5&type=share
https://bigfive.jr-genius.jp/ogp-image?code=31511&gender=F&rarity=r3
```

キャラ画像が新しいもの（背景透明・トリム済み）になっていればOK。

## トラブルシューティング

- **Puppeteerが動かない**: `npm install puppeteer` を実行。Edgeがインストールされている必要あり。
- **R2アップロードが失敗する**: スクリプト内の認証情報（Account ID / Access Key / Secret Key）が有効か確認。
- **Workerのデプロイは不要**: 今回は画像のみの更新。Workerのコードは変更なし。
- **generate_v2.js が途中で止まる**: resume対応なので再実行すれば続きから再開。ログを見てどこまで進んだか確認可能。
