## 生成スクリプト

生成は `_private/generate_blog_image.py` を使う。

```python
import openai, base64, os

client = openai.OpenAI(api_key="<APIキー>")

prompt = """<ここにプロンプトを貼る>"""

response = client.images.generate(
    model="dall-e-3",
    prompt=prompt,
    size="1792x1024",
    quality="hd",
    n=1,
    response_format="b64_json"
)

img_data = base64.b64decode(response.data[0].b64_json)
out_path = r"G:\マイドライブ\Claude Code\ウェブサイト\あそびラボ_プロジェクト\ビックファイブ診断サイト\images\blog\<ファイル名>.jpg"
with open(out_path, "wb") as f:
    f.write(img_data)
print("保存完了:", out_path)
```
