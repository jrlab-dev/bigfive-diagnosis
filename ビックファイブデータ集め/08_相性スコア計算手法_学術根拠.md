## 1. 最重要の前提：「相性の通説」を覆す研究結果

### 1-1. 「似た者同士が合う」は本当か？

| 通説 | 研究結果 |
|------|---------|
| 似た性格同士が合う（類似性仮説） | **規範的バイアスを除去すると効果はほぼゼロに消失（r=.21 → .04）** |
| 正反対が引き合う（補完性仮説） | 体系的な証拠なし |

**最も正確な答え：「似ているかどうか」より「自分・相手のNが低くAが高いかどうか」の方が関係満足度への影響が大きい。**

> Pronk, T. M., & Denissen, J. J. A. (2020). A dislike-based account of partner selection. *Social Psychological and Personality Science, 11*(5), 631–638.
> Wood, D., & Furr, R. M. (2016). The correlates of similarity estimates are often misleadingly positive. *Personality and Social Psychology Review, 20*(2), 79–99.

## 2. 因子別の関係満足度への影響（メタ分析数値） 🔵 実測

### 2-1. Actor効果（自分の性格が自分の満足度に与える影響）

> 出典：40研究・7,640名を対象にしたメタ分析

| 順位 | 因子 | ρ（補正済み相関） | 方向 |
|------|------|----------------|------|
| **1位** | **N（神経症傾向）** | **−.29** | 高いほど不満足 |
| **2位** | **A（協調性）** | **+.29** | 高いほど満足 |
| **3位** | C（誠実性） | +.25 | 高いほど満足 |
| **4位** | E（外向性） | +.17 | 高いほど満足 |
| **5位** | O（開放性） | +.10 | ほぼ無関係 |

### 2-2. Partner効果（相手の性格が自分の満足度に与える影響）

> 出典：Malouff et al. (2010)、19サンプル・3,848名のメタ分析

| 因子 | r（相関係数） | 95%CI |
|------|-------------|-------|
| **N（神経症傾向）** | **−.22** | [−.26, −.19] |
| **A（協調性）** | **.15** | [.11, .20] |
| C（誠実性） | .12 | [.09, .16] |
| E（外向性） | .06 | [.01, .10] |
| O（開放性） | .03（非有意） | [−.03, .09] |

**実装上の重要示唆：**
- 相性スコアは「2人の差」だけでなく**「2人の絶対値（レベル）」**で計算する必要がある
- OはActor・Partner効果ともに弱いため、相性への重みを最小にする

## 3. 因子間の交互作用（組み合わせ効果） 🔵 実測（N×A組み合わせのみ）/ 🟡 推定（その他）

### 3-1. N × A の組み合わせ（最重要）

| 組み合わせ | 関係への影響 | 評価 |
|-----------|------------|------|
| **N低 + A高** | 情緒安定 × 協調的。関係満足度が最高水準 | 最良 ★★★★★ |
| **N低 + A低** | 情緒は安定しているが、協力場面で摩擦 | 普通 ★★★☆☆ |
| **N高 + A高** | 感情的だが協調しようとする努力が補償 | 普通 ★★★☆☆ |
| **N高 + A低** | 感情爆発 × 非協力。衝突頻度最大 | 最悪 ★☆☆☆☆ |

> 根拠：Kern & Friedman (2008); Journal of Youth and Adolescence (2024)

### 3-2. ポジティビティ効果（Bleidorn et al., 2018）

**5研究（n=60〜91カップル）の一貫した発見：**
- 距離（差）の直接効果：β = −.29〜−.77
- ポジティビティ（各自の特性の高さ）：β = +.42〜+.66
- → **「差の大きさ」と「各人の特性レベルの高さ」の両方が相性に影響**

## 4. 類似性計算の注意点（D²スコアの問題）

### 4-1. NDC問題（規範的望ましさ交絡）

単純なユークリッド距離（D²）やプロファイル相関（Q相関）には深刻な問題がある：

**問題のメカニズム：**
```
「望ましい特性を持つ人（N低・A高）は関係が良い」
           ↓
「2人の特性が似ている（どちらも望ましい）と距離が小さい」
           ↓
「距離が小さい（類似している）カップルは関係が良い」← これは偽の因果関係
```

**数値で見る問題：**
- 生のD²による類似性効果：r = .21
- 規範性（集団平均への類似度）を統制後：r = .04（ほぼ消失）

### 4-2. サイト実装での対応方針

**採用するアプローチ：**
1. **Actor効果** = 自分のN・A・C・E・Oの絶対値を使う
2. **Partner効果** = 相手のN・A・C・E・Oの絶対値を使う
3. **差による補正** = Cのばらつき（生活スタイル）・Eのばらつき（社交スタイル）のみ使用
4. Oは差・絶対値ともに最小の重みにとどめる

## 5. 相性スコア計算式（学術根拠版） 🟡 推定（係数は🔵実測データから設計）

### 5-1. 設計方針

| 要素 | 計算方法 | 根拠 |
|------|---------|------|
| 自分のNの低さ | (6 − 自分N) × 5.0 | Actor効果 ρ = −.29 |
| 相手のNの低さ | (6 − 相手N) × 4.0 | Partner効果 r = −.22 |
| 自分のAの高さ | 自分A × 5.0 | Actor効果 ρ = +.29 |
| 相手のAの高さ | 相手A × 3.5 | Partner効果 r = +.15 |
| 自分のCの高さ | 自分C × 3.0 | Actor効果 ρ = +.25 |
| 相手のCの高さ | 相手C × 1.5 | Partner効果 r = +.12 |
| CスコアのΔ補正 | −ΔC × 2.0 | 生活スタイルミスマッチ |
| EスコアのΔ補正 | −ΔE × 1.5 | 社交スタイルミスマッチ |
| OスコアのΔ補正 | −ΔO × 0.5 | 効果量最小 |

### 5-2. 生スコア（0〜100点）の正規化計算

```
// 各変数の最大値
// 自分N低: (6-1)×5.0 = 25
// 相手N低: (6-1)×4.0 = 20
// 自分A: 5×5.0 = 25
// 相手A: 5×3.5 = 17.5
// 自分C: 5×3.0 = 15
// 相手C: 5×1.5 = 7.5
// ΔC補正: 最大ペナルティ = -4×2.0 = -8
// ΔE補正: 最大ペナルティ = -4×1.5 = -6
// ΔO補正: 最大ペナルティ = -4×0.5 = -2

// 最大生スコア = 25+20+25+17.5+15+7.5 = 110
// 最小生スコア = 0+0+0+0+0+0 + (-8-6-2) = -16

function calcCompatibility(myType, partnerType) {
  const [oM, cM, eM, aM, nM] = myType.split('').map(Number);
  const [oP, cP, eP, aP, nP] = partnerType.split('').map(Number);

  const dO = Math.abs(oM - oP);
  const dC = Math.abs(cM - cP);
  const dE = Math.abs(eM - eP);

  // 各要素のスコア
  const rawScore =
    (6 - nM) * 5.0 +   // 自分のN低さ（Actor効果）
    (6 - nP) * 4.0 +   // 相手のN低さ（Partner効果）
    aM * 5.0 +          // 自分のA高さ（Actor効果）
    aP * 3.5 +          // 相手のA高さ（Partner効果）
    cM * 3.0 +          // 自分のC高さ（Actor効果）
    cP * 1.5 +          // 相手のC高さ（Partner効果）
    (-dC * 2.0) +       // CスコアΔペナルティ
    (-dE * 1.5) +       // EスコアΔペナルティ
    (-dO * 0.5);        // OスコアΔペナルティ

  // 正規化: -16〜110 → 0〜100
  const MIN = -16, MAX = 110;
  return Math.round(Math.max(0, Math.min(100, (rawScore - MIN) / (MAX - MIN) * 100)));
}
```

### 5-3. 追加補正（特殊パターン）

| 条件 | 補正 | 根拠 |
|------|------|------|
| 自分N≥4 かつ 相手A≤2 | −15 | 最悪のN高×A低組み合わせ |
| 自分N≤2 かつ 相手N≤2 | +8 | 両者安定型・最安定パターン |
| 自分A≥4 かつ 相手A≥4 | +8 | 双方高協調・最良パターン |

## 7. 相性コメントの自動生成ロジック

### 7-1. 主要因子の特定

```javascript
function getCompatibilityComment(myType, partnerType, score) {
  const [oM,cM,eM,aM,nM] = myType.split('').map(Number);
  const [oP,cP,eP,aP,nP] = partnerType.split('').map(Number);

  let comments = [];

  // N（最重要）
  if (nM >= 4 && nP >= 4) {
    comments.push("お互い感情の波が大きく、ぶつかりやすい面があります。");
  } else if (nM <= 2 && nP <= 2) {
    comments.push("2人とも精神的に安定しています。困難な状況でも冷静に向き合えます。");
  } else if (nP <= 2 && nM >= 4) {
    comments.push("相手の安定感があなたの心を落ち着かせてくれます。");
  }

  // A
  if (aM >= 4 && aP >= 4) {
    comments.push("思いやりが深い2人。対立しても話し合いで解決できます。");
  } else if (aM <= 2 && aP <= 2) {
    comments.push("2人とも自己主張が強め。意見の違いが摩擦になることも。");
  }

  // C（生活スタイル）
  const dC = Math.abs(cM - cP);
  if (dC >= 3) {
    comments.push("計画性と自由さに大きな差があります。生活スタイルの摺り合わせが必要です。");
  }

  // E（社交スタイル）
  const dE = Math.abs(eM - eP);
  if (dE >= 3) {
    comments.push("社交的なニーズに差があります。休日の過ごし方の好みが異なる可能性があります。");
  }

  return comments.join(' ');
}
```
