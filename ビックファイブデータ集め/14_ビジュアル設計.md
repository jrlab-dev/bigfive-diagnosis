## 1. カラーシステム

### 1-1. 5因子カラー

各因子に色と「感情イメージ」を割り当てる。

| 因子 | 色名 | HEX | RGB | イメージ |
|------|------|-----|-----|---------|
| **O 開放性** | パープル | `#8b5cf6` | 139,92,246 | 想像力・宇宙・神秘 |
| **C 誠実性** | ブルー | `#3b82f6` | 59,130,246 | 信頼・秩序・海 |
| **E 外向性** | アンバー | `#f59e0b` | 245,158,11 | 活力・太陽・エネルギー |
| **A 協調性** | エメラルド | `#10b981` | 16,185,129 | 自然・調和・癒し |
| **N 神経症傾向** | ローズ | `#f43f5e` | 244,63,94 | 感情・情熱・感受性 |

> N因子の色は「悪い」イメージではなく「感情の豊かさ」として使う

## 8. OGP画像 設計（Canvas API）

SNSシェア時に表示される画像。Canvas APIでリアルタイム生成。

### 8-1. サイズ・レイアウト

```
サイズ：1200 × 630px

┌─────────────────────────────────────────────┐
│  [グラデーション背景：タイプカラー]           │
│                                             │
│   ビッグファイブ性格診断                      │ ← 小さいテキスト
│                                             │
│   「落ち着いた縁の下の力持ち」                │ ← 大きいテキスト
│                                             │
│   O ████████░░  E ████░░░░░░               │
│   C ██████████  A ████████░░               │
│             N ████░░░░░░                   │ ← 因子バー5本
│                                             │
│   #34251          jrlab-dev.github.io       │ ← コードとURL
└─────────────────────────────────────────────┘
```

### 8-2. Canvas生成コード（概略）

```javascript
function generateOGP(typeCode, typeName) {
  const canvas = document.createElement('canvas');
  canvas.width = 1200;
  canvas.height = 630;
  const ctx = canvas.getContext('2d');

  // 背景グラデーション（タイプカラーから）
  const grad = ctx.createLinearGradient(0, 0, 1200, 630);
  grad.addColorStop(0, getCoreTypeColor(typeCode));
  grad.addColorStop(1, '#0f172a');
  ctx.fillStyle = grad;
  ctx.fillRect(0, 0, 1200, 630);

  // タイプ名
  ctx.font = 'bold 72px "Noto Sans JP"';
  ctx.fillStyle = '#ffffff';
  ctx.textAlign = 'center';
  ctx.fillText(`「${typeName}」`, 600, 260);

  // 因子バー5本
  drawFactorBars(ctx, typeCode, 200, 340);

  // コード＆URL
  ctx.font = '28px "Inter"';
  ctx.fillStyle = '#94a3b8';
  ctx.textAlign = 'left';
  ctx.fillText(`#${typeCode}`, 80, 580);
  ctx.textAlign = 'right';
  ctx.fillText('jrlab-dev.github.io/asobi-lab/bigfive', 1120, 580);

  return canvas.toDataURL('image/png');
}
```
