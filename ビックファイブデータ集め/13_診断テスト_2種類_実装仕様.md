## A-1. 設計方針

🟡 推定（TIPI-J・小塩ら 2012 をベースに独自アレンジ）

- 各因子2問（正方向1問＋逆方向1問）
- 回答形式：**5段階ボタン**（1〜5）
- スコア計算：2問の平均 → **3段階** に変換
- 「とっつきやすさ」最優先。難しい言葉ゼロ

### 3段階変換テーブル

| 2問平均スコア | 段階 | 表示 |
|-------------|------|------|
| 1.0〜2.4 | 低（L） | ○○：低め |
| 2.5〜3.5 | 中（M） | ○○：ふつう |
| 3.6〜5.0 | 高（H） | ○○：高め |

→ 5因子 × 3段階 = **3⁵ = 243パターン**

## A-3. スコア計算式

```javascript
// 短縮版（10問）スコア計算
function calcShortTest(answers) {
  // answers = [Q1, Q2, Q3, Q4, Q5, Q6, Q7, Q8, Q9, Q10]  (各1〜5)

  const scores = {
    E: (answers[0] + (6 - answers[1])) / 2,
    A: (answers[2] + (6 - answers[3])) / 2,
    C: (answers[4] + (6 - answers[5])) / 2,
    N: (answers[6] + (6 - answers[7])) / 2,
    O: (answers[8] + (6 - answers[9])) / 2,
  };

  // 3段階変換
  const level = (score) => score <= 2.4 ? 1 : score <= 3.5 ? 3 : 5;
  // ※ 1=低, 3=中, 5=高（3125システムとの互換のため奇数値を使用）

  return {
    O: level(scores.O),
    C: level(scores.C),
    E: level(scores.E),
    A: level(scores.A),
    N: level(scores.N),
    code: `${level(scores.O)}${level(scores.C)}${level(scores.E)}${level(scores.A)}${level(scores.N)}`
    // 例："35351" （中・低・中・低・中ではなく、1/3/5のみ）
  };
}
```

> 短縮版の結果コードは 1/3/5 のみの5桁（例：13531）。
> 詳細版の結果と同じURL体系で扱える。
