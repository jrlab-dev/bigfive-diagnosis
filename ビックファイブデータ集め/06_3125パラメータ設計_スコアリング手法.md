## 1. 3125という数字の根拠

ビッグファイブの5因子（O・C・E・A・N）をそれぞれ **5段階** で評価することで：

**5 × 5 × 5 × 5 × 5 = 5⁵ = 3,125 パターン**

これがすべての性格パターンを網羅する最小単位となる。

---

## 2. 各因子のスコアリング設計 🟡 推定

### 2-1. 5段階スコアの定義

各因子は **1〜5の5段階** で評価する。

| スコア | 呼称 | 説明 |
|-------|------|------|
| **1** | 非常に低い | その特性がほぼ見られない |
| **2** | やや低い | 平均より低め |
| **3** | 中程度 | 平均的・バランス型 |
| **4** | やや高い | 平均より高め |
| **5** | 非常に高い | その特性が強く現れる |

### 2-2. 診断テストのスコア→段階変換

診断テスト（各因子6問、1〜5点）でのスコア合計 6〜30点を5段階に変換：

| 合計スコア | 段階 | 百分率ライン（正規分布近似） |
|-----------|------|---------------------------|
| 6〜11点 | 1（非常に低い） | 〜10% |
| 12〜16点 | 2（やや低い） | 10〜30% |
| 17〜23点 | 3（中程度） | 30〜70% |
| 24〜28点 | 4（やや高い） | 70〜90% |
| 29〜30点 | 5（非常に高い） | 90%〜 |

> 参考：日本人サンプル（Yoshino et al., 2022）の分布に合わせて閾値を微調整予定

## 3. タイプコード表記システム 🟠 創作

### 3-1. 基本表記

```
O[1-5]-C[1-5]-E[1-5]-A[1-5]-N[1-5]
```

例：`O3-C4-E2-A5-N1` = 開放性3・誠実性4・外向性2・協調性5・神経症傾向1

### 3-2. 短縮コード（URLパラメータ用）

5桁の数字で表現：`34251` = O3-C4-E2-A5-N1

## 4. 3125タイプの全パターン構造

### 4-1. 分布設計

完全ランダムの場合、各パターンは 1/3125 ≈ 0.032%。
実際の人口分布は正規分布に近いため、中央値（3-3-3-3-3）付近に集中する。

**頻出パターン上位（理論値）：**

| ランク | コード | 理論出現率 |
|--------|--------|-----------|
| 1 | 33333 | 最頻値・平均的性格 |
| 2〜6 | 33334 / 33332 / 33343... | 1因子のみ異なる |

### 4-2. 極端なタイプ

| コード | 通称（仮） | 特徴 |
|--------|-----------|------|
| 55555 | 「完璧人間型」 | 全因子最高。実際には存在稀 |
| 11111 | 「引きこもり型」 | 全因子最低。精神健康リスク高 |
| 55551 | 「ストレス耐性型」 | N低＋全他高 |
| 11115 | 「不安型天才」 | N高のみ、他低 |
| 55511 | 「社交的クリエイター」 | O高C高E高、A低N低 |

## 5. タイプ説明文の自動生成ロジック 🟠 創作

### 5-1. テンプレート構造

各タイプの説明文は **5因子のスコアを組み合わせたテンプレート** で生成する。

```
[主ドライバー因子]が[レベル表現]で、
[補助因子1]と[補助因子2]の組み合わせにより、
[行動傾向の説明]。
[特定のシーン]では[具体的な行動]が見られる傾向がある。
```

### 5-2. 因子スコアごとの記述フラグメント

#### O（開放性）記述フラグメント

| スコア | 短文フラグメント |
|-------|---------------|
| 1 | 慣れ親しんだことを大切にし、変化よりも安定を好む |
| 2 | どちらかといえば保守的で、実際的なアプローチを取る |
| 3 | 新しいことへの関心と安定へのニーズのバランスが取れている |
| 4 | 好奇心が旺盛で、新しいアイデアや体験を積極的に求める |
| 5 | 豊かな想像力と強い知的好奇心で、常に新しい視点を探求する |

#### C（誠実性）記述フラグメント

| スコア | 短文フラグメント |
|-------|---------------|
| 1 | 自由な流れに任せて行動し、計画よりも直感を重視する |
| 2 | 柔軟性を大切にし、細かい計画より臨機応変を好む |
| 3 | 計画性と柔軟性をバランスよく使い分ける |
| 4 | 責任感が強く、目標に向けて着実に取り組む |
| 5 | 高い自己規律と計画性で、何事も確実にやり遂げる |

#### E（外向性）記述フラグメント

| スコア | 短文フラグメント |
|-------|---------------|
| 1 | 一人の時間を大切にし、深い内省と集中を好む |
| 2 | 内向的で、少数の親しい関係を深めることを重視する |
| 3 | 社交的な面と一人の時間のニーズをバランスよく持つ |
| 4 | 社交的で、人との交流からエネルギーを得る |
| 5 | 活発で社交的、賑やかな場や新しい出会いを積極的に求める |

#### A（協調性）記述フラグメント

| スコア | 短文フラグメント |
|-------|---------------|
| 1 | 自己主張が明確で、競争的な場面でも主張を崩さない |
| 2 | 独立心が強く、自分の判断と利益を重視する傾向がある |
| 3 | 他者への配慮と自己主張のバランスが取れている |
| 4 | 思いやりがあり、周囲との調和を大切にする |
| 5 | 深い共感力と利他性を持ち、他者のために動くことを自然に行う |

#### N（神経症傾向）記述フラグメント

| スコア | 短文フラグメント |
|-------|---------------|
| 1 | 精神的な安定感が高く、ストレス下でも冷静さを保てる |
| 2 | 感情的な揺れが少なく、落ち着いて物事を判断できる |
| 3 | 感情の波はあるが、自分でコントロールできる範囲内 |
| 4 | 感情豊かで繊細。状況によって精神的に揺れることがある |
| 5 | 感情に敏感で深く、ストレスや不安を強く感じやすい |

> ※ N高の表現は「弱さ」でなく「感情の豊かさ・繊細さ」として中性的に表現する（ゲーム設計の前提と合わせる）

## 6. 3125タイプのグループ分類（サマリーシステム） 🟠 創作

3125タイプを少数の「大分類グループ」に整理することで、ユーザーが自分のタイプを把握しやすくする。

### 6-1. 主因子による16グループ分類（仮）

主導的な因子（最高スコアの因子）+ Nスコア（高/低）で大まかに分類。

| グループ | 主因子 | Nスコア | 仮称 |
|---------|--------|---------|------|
| A | O高 | 低 | 「安定した探求者」 |
| B | O高 | 高 | 「感情豊かなクリエイター」 |
| C | C高 | 低 | 「着実な実行者」 |
| D | C高 | 高 | 「完璧主義の頑張り屋」 |
| E | E高 | 低 | 「陽気なリーダー」 |
| F | E高 | 高 | 「熱量あふれる盛り上げ役」 |
| G | A高 | 低 | 「穏やかな調停者」 |
| H | A高 | 高 | 「傷つきやすい優しい人」 |

### 6-2. 主要5大タイプ（SEO・SNSシェア用）

より覚えやすい5つの代表タイプ：

| タイプ名 | コードの傾向 | キャッチコピー |
|---------|-----------|--------------|
| **探求者型** | O高・E中 | 「世界は発見で満ちている」 |
| **実行者型** | C高・N低 | 「有言実行が信条」 |
| **社交家型** | E高・A高 | 「人と一緒にいると輝く」 |
| **調停者型** | A高・N中 | 「みんなが笑顔でいられるように」 |
| **内省者型** | E低・O高 | 「深く考えることが力になる」 |
