# ビッグファイブ 有名人・キャラクタータイプ推定

> 作成日：2026年3月29日
> 用途：サイトのSEOコンテンツ・検索流入・共感訴求

---

## 信頼度マークについて

| マーク | 意味 |
|--------|------|
| 🔵 実測 | 論文・メタ分析の直接数値 |
| 🟡 推定 | 学術モデルから導出した値 |
| 🟠 創作 | サイト用に設計した名称・説明文 |

**重要な注記：**
- 🔵 実測：歴史的人物を対象にした学術研究（Simonton等の歴史心理学）で推定されたもの
- 🟡 推定：伝記・行動記録から専門家が推定したもの
- 🟠 創作：フィクションキャラクター、または一般的なイメージから独自に設定したもの

## 4. タイプ別「この有名人っぽい」コンテンツ設計

🟠 創作（サイトコンテンツ企画）

### 4-1. ページ設計案

```
タイプ詳細ページの末尾に追加：

【このタイプっぽい有名人・キャラクター】
実在の人物：○○、○○（あくまでイメージです）
アニメキャラ：○○（作品名）、○○（作品名）
```

### 4-2. SNS拡散を狙うコンテンツ例

```
「ビッグファイブで診断したら『黙々と結果を出す人』でした。
 これってゾロっぽくない？ #ビッグファイブ #性格診断」

「私のタイプ『縁の下の力持ち』は
 ハーミオーネよりハリーっぽいかも笑 #3125タイプ」
```

### 4-3. 「あなたは誰タイプ？」クイズ形式への展開

有名キャラの行動パターンを選択肢にして、最後にビッグファイブタイプを表示する「キャラ共感型診断」も展開可能。

## 注意事項（サイト掲載時）

1. **存命の著名人** → 断定せず「○○さんのイメージに近いタイプ」と表現
2. **歴史的人物** → 「研究者による推定」と注記
3. **フィクションキャラ** → 「○○っぽいタイプ」と表現（著作権に注意）
4. **全般** → 「あくまで参考・エンターテインメントとして楽しんでください」の免責を入れる
