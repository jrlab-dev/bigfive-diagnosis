# ビッグファイブ キャラクター 見た目ルール

## 基本方針
5因子のスコア（1〜5）に応じて、髪型・服装・表情・色・ポーズが変わる。
デザインスタイル（2.5頭身・顔のパーツ等）は全キャラ共通で固定。

---

## 各因子の担当要素

| 因子 | 主な担当 | 補助 |
|---|---|---|
| **O（開放性）** | 髪型の個性度 | 服装の個性度 |
| **C（誠実性）** | 髪の整い度 | 服装のきちんと感 |
| **E（外向性）** | 色の鮮やかさ | 表情の明るさ・ポーズ |
| **A（協調性）** | 目の印象 | 色の温度（暖色/寒色） |
| **N（感受性）** | 眉の形 | 表情の豊かさ |

---

## O（開放性）— 髪型の個性度 ＋ 服装の個性度

### 髪型
| レベル | 髪型 |
|---|---|
| **O1** | シンプル・無難・普通的（ストレート、ベリーショートなど） |
| **O2** | ふつう＋少しだけ特徴あり |
| **O3** | 個性はあるが違和感はない（やや長め、アレンジあり） |
| **O4** | 個性的（メッシュ、アシンメトリー、独特の切り方） |
| **O5** | 奇抜・アート系（グラデーション、激しいアレンジ、特徴的な形） |

### 服装
| レベル | 服装 |
|---|---|
| **O1** | ベーシック（Tシャツ＋デニム等、無難） |
| **O2** | ふつうのカジュアル |
| **O3** | ふつう＋ワンポイント（柄あり、小物あり） |
| **O4** | 個性的なコーデ（レイヤード、独特の組み合わせ） |
| **O5** | 独自のスタイル（アート系、 Y2K系、サブカル系など） |

---

## C（誠実性）— 整い度 ＋ 服装のきちんと感

### 髪の整い度
| レベル | 髪の状態 |
|---|---|
| **C1** | ボサボサ・無造作・寝癖あり |
| **C2** | そこそこだが少しラフ |
| **C3** | 普通に手入れされている |
| **C4** | きちんとセットされている |
| **C5** | ぴしっと整っている・一髪乱れず |

### 服装
| レベル | 服装 |
|---|---|
| **C1** | ラフ・着崩し・シワあり・サイズ感自由 |
| **C2** | カジュアルだが少し気にしてる |
| **C3** | 普通の服装 |
| **C4** | きちんとしたカジュアル・アイロン済み |
| **C5** | フォーマル寄り・襟付き・アクセサリーあり |

---

## E（外向性）— 色の鮮やかさ ＋ 表情・ポーズ

### 色の鮮やかさ（髪色＋服の色）
| レベル | 色調 |
|---|---|
| **E1** | 暗め・くすみ系（黒、ネイビー、グレー、ダークブラウン） |
| **E2** | 少し暗め・落ち着いた色 |
| **E3** | 中間色（スタンダードな色） |
| **E4** | 明るめ・鮮やか |
| **E5** | 派手・原色系（赤、黄、ターコイズ、ピンクなど） |

### 表情
| レベル | 表情 |
|---|---|
| **E1** | 無表情・静か・口は閉じている |
| **E2** | ほんのり微笑み・控えめ |
| **E3** | ふつうの穏やかな表情 |
| **E4** | 笑顔・明るい |
| **E5** → 大きな笑顔・目を細めて笑う |

### ポーズ
| レベル | ポーズ |
|---|---|
| **E1** | 直立・腕は体に沿って下ろす |
| **E3** | ふつうの立ち姿 |
| **E5** | 少し動きがある（手を挙げる・前に出る等） |

---

## A（協調性）— 目の印象 ＋ 色の温度

### 目の印象
| レベル | 目の印象 |
|---|---|
| **A1** | 鋭い・強い瞳・少しつり目 |
| **A2** | ややクールな目 |
| **A3** | ふつう |
| **A4** | 優しい・柔らかい目 |
| **A5** | ぱっちりした優しい瞳・キラキラ感 |

### 色の温度
| レベル | 色の傾向 |
|---|---|
| **A1** | 寒色系（青、紫、黒、白） |
| **A3** | 中間 |
| **A5** | 暖色系（オレンジ、ピンク、クリーム、ベージュ） |

---

## N（感受性）— 眉の形 ＋ 表情の豊かさ

### 眉の形
| レベル | 眉 |
|---|---|
| **N1** | 平坦・穏やかな眉 |
| **N2** | ふつう |
| **N3** | 少し感情が出る眉 |
| **N4** | 動きのある眉・上げ眉 |
| **N5** | 困った眉・不安げな眉・または強い集中の眉 |

### 表情の豊かさ
| レベル | 表情 |
|---|---|
| **N1** | 全体的にリラックス・何も考えてなさそう |
| **N3** | ふつう |
| **N5** | 何か感情を抱えていそう・繊細な雰囲気 |

---

## 組み合わせ例

| タイプ | O | C | E | A | N | どんな見た目？ |
|---|---|---|---|---|---|---|
| ルフィ的 | 5 | 1 | 5 | 4 | 3 | 奇抜な髪・ボサボサ・派手な色・優しい目 |
| ゾロ的 | 1 | 4 | 2 | 2 | 1 | シンプル髪・整っている・暗め色・鋭い目 |
| フランキー的 | 4 | 2 | 5 | 5 | 3 | 個性的髪・ラフ・派手色・優しい目 |
| 冷静な軍師 | 3 | 5 | 1 | 2 | 1 | ふつう髪・ぴしっ・暗い色・鋭い目・リラックス |
