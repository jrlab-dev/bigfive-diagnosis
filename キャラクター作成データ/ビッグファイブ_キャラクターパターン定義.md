# ビッグファイブ キャラクターパターン定義

> 3125パターンの組み合わせ定義データ
> 顔125パターン（神経症傾向 × 外向性 × 協調性）

---

## 第1部：5要素 × 5段階の定義（25個）

### N：神経症傾向（顔で表現）— 1位：表情に出やすい

| レベル | ラベル | 顔の特徴 |
|---|---|---|
| N1 | とても安定 | 眉間なし、目が穏やか、口角が自然に上がっている、全体的にリラックスした顔 |
| N2 | やや安定 | ほっとしている顔、少し柔らかい目、口角がやや上がっている |
| N3 | 普通 | 中立的な表情、目も口も特に特徴なし、何の感情も読み取れない顔 |
| N4 | やや不安 | 眉間が少し寄っている、口角がやや下がる、目が不安そうに見える |
| N5 | とても不安 | 眉間に深いシワ、口角が下がる、目に不安と緊張が見える、全体的に苦しそう |

---

### E：外向性（顔で表現）— 2位

| レベル | ラベル | 顔の特徴 |
|---|---|---|
| E1 | とても内向的 | 目が伏せがち、目線が下や遠く、口角が閉じている、静かで控えめな顔 |
| E2 | やや内向的 | 目が小さめ、目線がやや下、口がへの字に近い、物静かな顔 |
| E3 | 普通 | 目は普通の大きさ、目線は正面、中立的な口元、特に目立たない顔 |
| E4 | やや外向的 | 目が少し大きく見える、口角が上がり気味、明るさを感じる顔 |
| E5 | とても外向的 | 目が大きく輝いている、口角が大きく上がった笑顔、エネルギーに満ちた活き活きした顔 |

---

### A：協調性（顔で表現）— 3位

| レベル | ラベル | 顔の特徴 |
|---|---|---|
| A1 | とても非協力的 | 目つきが鋭い、眉が下がっている（八の字）、口が引き結ばれている、近寄りがたい印象 |
| A2 | やや非協力的 | 目が少し細い、眉がやや下がっている、口元が固い、冷たい印象 |
| A3 | 普通 | 目も眉も特に特徴なし、口は普通の状態、どちらとも言えない印象 |
| A4 | やや協力的 | 目が柔らかい、眉が少し上がっている、口角がやや上がっている、優しい印象 |
| A5 | とても協力的 | 目がとても優しい、眉が柔らかく上がっている、口元が穏やかな微笑み、誰にでも好かれそうな印象 |

---

### O：開放性（ポーズで表現）

| レベル | ラベル | ポーズの特徴 |
|---|---|---|
| O1 | とても保守的 | 腕を組んでいる、体が縮こまっている、視線が下、閉じた姿勢、何かを守るような構え |
| O2 | やや保守的 | 片手がポケット、体がやや前かがみ、視線がやや下、控えめな立ち姿 |
| O3 | 普通 | 両手が自然に下に落ちている、体はまっすぐ、視線は正面、何の特徴もない立ち姿 |
| O4 | やや開放的 | 片手を少し上げている、体がやや前のめり、視線が上や遠く、興味を持っているような姿勢 |
| O5 | とても開放的 | 両腕を広げている、体が前に進み出している、視線が上を見上げている、何かを指さす、世界に向かって開かれた姿勢 |

---

### C：誠実性（ポーズで表現）

| レベル | ラベル | ポーズの特徴 |
|---|---|---|
| C1 | とても無頓着 | 猫背、肩が前に出ている、体が傾いている、だらっとした姿勢、ボタンが外れていても気にしなさそう |
| C2 | やや無頓着 | 少し猫背気味、肩が少し下がっている、体がやや傾いている、リラックスしすぎた姿勢 |
| C3 | 普通 | 背筋は普通、肩の位置も普通、体はまっすぐ立っているが硬すぎない、平均的な立ち姿 |
| C4 | やや几帳面 | 背筋がやや伸びている、肩がしっかり上がっている、体がきっちりと立っている、気配りのある姿勢 |
| C5 | とても几帳面 | 背筋がピンと伸びている、肩が真っ直ぐ、体が完璧にまっすぐ、全体的にきっちりと整った姿勢、隙がない |
