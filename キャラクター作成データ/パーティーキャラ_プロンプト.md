# パーティーキャラ 画像生成プロンプト

## 使い方
各キャラのプロンプトをそのままAIに貼り付ければOK。1回で男女2体が並んで生成されます。
生成後、Claude Codeに画像を半分に切ってもらってください。

## 生成時のコツ
- **参考画像必須：** プロンプトと一緒に `画像/決定モデル.png` を貼り付ける（テイスト・バランス維持のため）
- 参考画像なしではテイスト・バランスが崩れるため、必ず貼り付ける
- 「学校の制服ではない。学生ではない。」で制服化を防ぐ
- プロンプト内で髪色・服装を具体的に指定しているので服装はコピーされない

## 画像の保存名ルール
生成時（1枚に男女2体）：
```
char_001_自由人_O4C1E4A3N3.png
```
切り離し後：
```
char_001F_自由人_O4C1E4A3N3.png  （左・女性）
char_001M_自由人_O4C1E4A3N3.png  （右・男性）
```
保存先：`画像/パーティーキャラ/` フォルダ
