# ビッグファイブ性格診断テスト

心理学の「ビッグファイブ（Big Five）」性格特性理論に基づいた診断テストサイト。

## 特徴

- **科学的根拠**: 国際的に標準化されたビッグファイブ理論（5因子モデル）を採用
- **5つの性格因子**: 開放性(O)・誠実性(C)・外向性(E)・協調性(A)・神経症傾向(N)を測定
- **簡易テスト**: 大人向け20問・子ども向け15問の2種類
- **結果詳細**: 各特性のスコア＋適性職業のマッチング表示

## ビッグファイブ理論について

ビッグファイブ（Big Five）は、性格心理学において最も広く受け入れられている性格特性理論です。5つの因子で人間の性格を記述します：

1. **開放性 (Openness)**: 創造性、好奇心、知的興味
2. **誠実性 (Conscientiousness)**: 規律性、責任感、組織力
3. **外向性 (Extraversion)**: 社交性、エネルギー、活動性
4. **協調性 (Agreeableness)**: 共感性、協力性、信頼性
5. **神経症傾向 (Neuroticism)**: 感情安定性、ストレス耐性
